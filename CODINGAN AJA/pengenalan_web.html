Pengenalan Web

World Wide Web (WWW) adalah kumpulan halaman web di seluruh dunia yang terhubung menggunakan protokol HTTP dan dapat diakses melalui browser.

Struktur Dasar HTML

<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        Isi halaman di sini
    </body>
</html>

Kesimpulan

HTML digunakan untuk membuat halaman web yang terdiri dari elemen-elemen seperti teks, gambar, tautan, dan tabel. Semua elemen ini diatur dalam struktur <head> dan <body>.
